# Project Summary Report Format

## Metadata
- title: "Project Summary"
- max_words: 2500
- date_format: "YYYY-MM-DD"

## Section Weights
- Project Overview: 0.15
- Objectives & Scope: 0.15
- Current Status: 0.20
- Key Milestones: 0.15
- Stakeholders & Resources: 0.10
- Risks & Dependencies: 0.15
- Next Steps: 0.10

## Structure

### Project Overview
Project name, description, business context. Project type, start date, target completion. Sponsor and project lead.

### Objectives & Scope
Primary objectives with measurable success criteria. What is in scope vs out of scope. Any scope changes and rationale.

### Current Status
Overall health (GREEN/YELLOW/RED) with explanation. Progress percentage and comparison to plan. Budget status: allocated, spent, projected, variance. Schedule status: ahead/on-track/behind. Recent accomplishments. Current blockers and mitigation actions.

### Key Milestones
Milestone tracking: name, target date, actual date, status. Upcoming milestones with readiness assessment. Completed milestones and lessons learned.

### Stakeholders & Resources
Key stakeholders with interest and influence levels. Project team members and roles. Resource utilization and any gaps.

### Risks & Dependencies
Active risks: description, probability, impact, mitigation strategy, owner. Internal and external dependencies with status and impact if delayed.

### Next Steps
Immediate actions (1-2 weeks) with owners. Upcoming phase goals and deliverables. Pending decisions with deadlines. Communication schedule.
